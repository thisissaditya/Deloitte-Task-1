# Deloitte Australia Technology Virtual Experience Program – Task 1

## Task Overview

This project is my solution for **Task 1: Create an algorithm to unify two different data models** from the Deloitte Australia Technology Virtual Experience Program on Forage.

The objective was to convert two different IoT telemetry JSON formats into a single standardized output format while ensuring all provided unit tests pass successfully.

## Files Included

* `main.py` – Solution implementation
* `data-1.json` – Input format 1
* `data-2.json` – Input format 2
* `data-result.json` – Expected unified output format

## Solution Approach

The solution:

1. Detects the input JSON format.
2. Converts Format 1 into the required unified structure.
3. Converts Format 2 into the required unified structure.
4. Normalizes timestamps and location data.
5. Validates correctness using the provided unit tests.

## Test Results

All provided unit tests pass successfully.
